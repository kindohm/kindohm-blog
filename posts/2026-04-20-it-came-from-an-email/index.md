[This post](posts/2026-04-19/april-24-new-world/) came from an email.

This is a custom-rolled blog, built with [Eleventy](https://www.11ty.dev/).
It doesn't have a nice interface to create posts, because it is all
built from code. That means that I have to be _at my computer_ to share an
image here, or publish a little bit of text.

_Until this weekend._

Now I can send an email to a custom address at [postmarkapp](https://postmarkapp.com),
which invokes a nodejs app on a custom web server I run. The app inspects the
email content, extracts any images, and publishes a blog post in my
code repository, which deploys here.

The hardest part about this was getting back into Linux admin to host my
nodejs app. That probably took about 75% of the time!

But anyway, this all benefits you. Maybe I'll share more images and text
now that I have this capability.
